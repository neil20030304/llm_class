# Task 2 — LangGraph Agent Frameworks

Building a multi-agent chat system step by step using LangGraph, from a simple single-model agent to a crash-recoverable multi-model conversation system.

## Files

| File | Description |
|---|---|
| `langraph_llama_agent.py` | Combined agent with verbose/quiet tracing, empty-input guard, Hey Qwen routing (tasks 1–4) |
| `lg_graph.png` | LangGraph visualization generated by Mermaid |

## Portfolio

See [`Topic2Frameworks/README.md`](Topic2Frameworks/README.md) for the full write-up and all individual task versions.

## Quick Start

```bash
conda activate llm-class
pip install langgraph langchain-huggingface transformers torch accelerate
huggingface-cli login

python langraph_llama_agent.py
```

**Runtime commands:**
- `verbose` — enable per-node `[TRACE]` output
- `quiet` — disable tracing
- `Hey Qwen <message>` — route to Qwen model
- `quit` / `exit` — exit
